# AI 政委

一个 Windows 本地优先的专注监督原型。它只在你主动开始专注会话后运行：

- 每 5 秒读取当前前台窗口标题和进程名。
- 用你设置的专注词 / 分心词做本地判断。
- 连续偏离时依次通知、要求报到、覆盖屏幕 20 秒。
- 可选用 OpenAI 视觉模型查看屏幕，最多每 60 秒一次。
- 截图不落盘；日志只保存窗口信息、判断结果和时间。
- `Ctrl + Shift + F12` 永远可以紧急停止。

## 运行

需要 Node.js 22 或更新版本。

```powershell
npm.cmd install
npm.cmd start
```

若要开启 AI 判断，先在启动应用的终端中设置：

```powershell
$env:OPENAI_API_KEY="你的 API key"
npm.cmd start
```

即使环境中存在 API key，AI 功能也默认关闭，必须在界面中手动勾选。

## 使用建议

专注关键词应写具体工具或项目名，例如 `vscode, figma, 毕业论文`。分心关键词应写你自己最常打开的站点或应用，例如 `bilibili, steam`。规则命中时不会调用 AI。

这不是家长控制或雇员监控工具，不应安装到他人的电脑，也不应隐藏运行。它不能替代睡眠、休息、心理咨询或对工作量本身的调整。
