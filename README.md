# ОГАС政委

一个 Windows 本地优先的专注监督原型。它只在你主动开始专注会话后运行：

- 每 5 秒读取当前前台窗口标题和进程名。
- 用你设置的专注词 / 分心词做本地判断；微信、企业微信、QQ、钉钉等中文关键词会匹配常见英文进程名。
- 默认自动识别 Steam、Epic、Xbox、GOG、EA、Ubisoft、Riot 和 Battle.net 游戏及客户端。
- 偏离时先警告；再次偏离才扣分，严重持续偏离时会升级提醒。
- 首次偏离时通过所配置兼容 API 的语音模型发出提醒并缓存音频。
- 可选 `Onyx`（浑厚威严）、`Echo`（冷静克制）或 `Ash`（干练锐利）音色。
- 可选用支持图片输入的 OpenAI-compatible 模型查看屏幕，最多每 60 秒一次。
- 本地规则无法判断时，先把进程、标题和路径交给文字模型；只有文字模型拿不准才截图。
- 明确的文字判断缓存 30 分钟，减少重复调用和不必要的截图。
- 屏幕视觉清晰度可选“标准”或“高清”；高清按显示器缩放比例采集，最长边限制为 2560 像素并使用高细节分析。截图不落盘。
- `Ctrl + Shift + F12` 永远可以紧急停止。
- 完成专注轮次按每完整 5 分钟奖励 1 点，未满 5 分钟的余数不计；并按苏军军衔从列兵晋升至元帅。
- 每天可输入待办并由 AI 整理为完成标准清晰的每日计划；当天可继续追加新目标，旧项目和完成状态不会被覆盖，重复标题会被忽略，每日最多 12 项。计划按本地日期自动清零，每项可用文字或粘贴截图交给 AI 审核，通过后奖励 1 点，已完成项不能重复领奖。
- 每日计划提醒可选开启并自行设置时间；提醒只提示制定计划，不会强制进入专注。
- 首次判断偏离只警告，警告期间再次偏离才扣 1 点并重置；累计 5 次“在推进”可清除警告。积分没有负数下限；积分低于 0 或 60 分钟内累计扣 5 点会进入惩戒营。
- 人格 Prompt 可在界面中直接编辑，控制提醒措辞、态度和语音风格。
- 专注词、分心词、时长、模型、语音、Cold Turkey 和娱乐模式等表单设置会自动保存，并在下次启动时恢复。
- 提前结束需要提交具体完成证据，由文字或视觉模型审查。
- 关闭窗口、强行停止或使用紧急快捷键会扣除 3 点，但始终保留安全退出能力。
- 强制结束专注时会立即公布本轮 Cold Turkey 密码，只扣除强退本身的 3 点，不再要求通过异常恢复额外扣分。
- Cold Turkey Pro 是可选增强，不安装也可以使用专注计时、关键词监督、AI 判断、每日计划、荣誉值和娱乐津贴。若已安装，可联动 Cold Turkey Pro：每轮生成随机密码并锁定指定 block，应用按专注倒计时记录公布时间，完成后显示密码。当前官方 CLI 未提供由本应用可靠写入自动结束时间的参数，因此使用密码锁而非 Cold Turkey 定时计划。
- 专注密码公布后必须确认已在 Cold Turkey 中实际解除 Block，确认前禁止创建下一把专注密码，避免应用密码与 Cold Turkey 当前密码错位。
- 若旧版本已造成密码错位，可使用 Cold Turkey 状态区的“事故恢复：尝试上一份密码”，从加密保险库读取前一份专注密码。
- 进入惩戒营时会用独立密码强制启动 Cold Turkey `Games` Block；只有积分恢复到 0 且惩戒时限结束、身份恢复正常后才公布密码。
- 惩戒营期间每 30 秒通过 Cold Turkey CLI 检查 `Games` 的真实启用状态；若明确检测到 `Disabled`，应用会立即生成新密码并重新锁定。若 Cold Turkey 主界面导致状态查询无输出，则使用当前加密密码重新发送锁定命令，避免密码错位。
- Cold Turkey 密码使用 Windows 安全存储加密；异常恢复密码会扣除 3 点。
- 最近 5 次成功使用的 Cold Turkey 密码会额外保存在隐藏、限权且加密的本地恢复保险库中，新记录会自动淘汰最旧记录。
- 不再生成根目录明文恢复密码文件；自动重锁后，旧历史密码不能代表当前锁，当前密码只从对应的加密会话记录读取。
- 可按 3 至 60 分钟间隔查看一次屏幕，生成不羞辱人的“战地点评”通知。
- 专注中首次判断偏离只发出警告；警告未清除前再次偏离才扣 1 点并重置。累计 5 次“在推进”也会清除当前警告。
- 娱乐津贴统一按 `1 荣誉点 = 5 分钟`兑换，不再提供免费开启；津贴使用记录按本地日期每日清零。
- 周一至周五需先自然完成累计 180 分钟专注才能兑换。工作日津贴随每一级军衔递增：列兵从 60 分钟起，普通晋升通常增加 5 分钟；少尉、少校、少将有 15 分钟跃升，元帅达到最高 180 分钟。惩戒营不能使用津贴。
- 惩戒营仍可开启 24 小时娱乐限制，但必须使用与惩戒营 `Games` 锁不同的 Cold Turkey Block。
- 周末津贴没有每日时长上限和专注门槛，但每次娱乐仍需消耗荣誉点。
- 娱乐模式可设置 15 至 600 秒的观察间隔。
- 娱乐模式可单独关闭 AI 点评；关闭后仍可计时和使用津贴，但不会截图、调用视觉模型或播放点评语音。
- 安装 Cold Turkey 后，可开启持续 24 小时的娱乐限制：Cold Turkey 使用随机密码锁定指定 Block，期间使用娱乐津贴需要按既定比例消耗荣誉点。
- 兑换开始时显示当前解锁密码；兑换时间结束或手动停止后，自动生成新密码并重新锁定。Block 需要预先包含目标游戏和娱乐网站。
- 24 小时限制状态会持久化；应用异常退出后，下次启动会尝试恢复锁定。提前恢复密码按异常中止扣除 3 点。
- 每次娱乐模式会话保留最近 30 轮临时上下文，使点评连贯并可适时给出建议；停止后立即清除。
- 识别到游戏和局势时，娱乐模式会更主动地给出走位、技能、配装、资源或团队协作等战术建议；无法确认时不会硬猜。
- AI 语音可在界面中按 `0.75×` 至 `1.5×` 调整语速，默认 `1.1×`。
- 娱乐点评只通过 AI 语音播报，不弹系统通知，也不写入“近期观察”或活动日志。
- 娱乐模式开启后会等待一个完整观察间隔再首次查看屏幕，避免误读应用自身界面。
- 支持任意提供 OpenAI-compatible `/v1/chat/completions` 的服务，可在界面配置服务名称、Base URL、API Key 和模型名。
- API Key 只通过界面输入，并使用 Electron 系统安全存储加密；保存后输入框立即清空且不会回显，也不会写入普通设置文件。
- 视觉需要服务商和模型支持 `image_url` 图片消息；语音需要服务商实现 `/v1/audio/speech` 并能返回 WAV。
- 可将文字判断、完成证据审查、提醒措辞和视觉任务优先分流给本地 Ollama。
- Ollama 模型不可用时可选择回退所配置的兼容 API。
- 界面会显示最近一次文字/视觉调用和语音调用实际使用的提供方、模型，以及是否由 Ollama 回退到 OpenAI。

## 运行

需要 Node.js 22 或更新版本。

最方便的方法是双击项目根目录中的 `Start AI Commissar.cmd`。第一次运行会自动安装依赖。

也可以在 PowerShell 中运行：

```powershell
npm.cmd install
npm.cmd start
```

直接在界面的“通用 OpenAI 格式 API”中填写服务名称、Base URL、API Key、文字/视觉模型和可选语音模型。API Key 保存后会加密存储并从输入框中清除。即使已经配置 API Key，屏幕 AI 判断也必须在界面中手动勾选。

若要使用本地 Ollama，先安装生成模型，例如：

```powershell
ollama pull qwen3:8b
ollama pull qwen3-vl:8b
```

启动 Ollama 后，在界面的“本地 Ollama 分流”中启用并填写模型名。文字模型负责进程判断、完成证据审查和提醒措辞；视觉模型负责看屏幕及战地点评。

## 使用建议

专注关键词应写具体工具或项目名，例如 `vscode, figma, 毕业论文`。分心关键词应写你自己最常打开的站点或应用，例如 `bilibili, steam`。规则命中时不会调用 AI。

这不是家长控制或雇员监控工具，不应安装到他人的电脑，也不应隐藏运行。它不能替代睡眠、休息、心理咨询或对工作量本身的调整。
