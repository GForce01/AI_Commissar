# ОГАС政委

一个 Windows 本地优先的专注辅助应用。它在你主动开启专注后读取前台窗口标题和进程名，用关键词、本地规则、可选 AI、可选 Cold Turkey 联动来提醒你回到任务。

它不是监控软件，也不应该安装到别人的电脑上。它的设计目标是：让你更容易遵守自己定下的计划，同时保留随时停止和恢复的主动权。

## 快速开始

需要 Node.js 22 或更新版本。

最简单的启动方式是双击项目根目录中的 `Start AI Commissar.cmd`。第一次启动会自动安装依赖。

也可以在 PowerShell 中运行：

```powershell
npm.cmd install
npm.cmd start
```

第一次使用建议先不配置 AI 和 Cold Turkey：

1. 写下当前任务。
2. 填写专注词，例如 `vscode, unity, 论文, github`。
3. 填写分心词，例如 `bilibili, steam, 微信`。
4. 设置时长，点击“开始专注”。

更完整的操作说明见 [USAGE.md](USAGE.md)。

## 核心能力

**专注监督**

- 每 5 秒读取当前前台窗口标题和进程名。
- 用专注词 / 分心词做本地判断；微信、企业微信、QQ、钉钉等中文关键词会匹配常见英文进程名。
- 默认自动识别 Steam、Epic、Xbox、GOG、EA、Ubisoft、Riot 和 Battle.net 游戏及客户端。
- 首次判断偏离只警告；警告期间再次偏离才扣 1 点并重置。
- 警告期间累计 5 次“在推进”会清除当前警告。
- `Ctrl + Shift + F12` 永远可以紧急停止。

**每日计划**

- 每天输入待办，由 AI 整理为完成标准清晰的计划项。
- 当天可继续追加新目标，不覆盖旧项目和完成状态。
- 每日最多 12 项，本地日期变化后自动清零。
- 每项可用文字或粘贴截图提交证据，审核通过奖励 1 荣誉值。

**荣誉值与娱乐津贴**

- 完成专注按每完整 5 分钟奖励 1 点，余数不计。
- 娱乐津贴按 `1 荣誉点 = 5 分钟`兑换。
- 工作日需要当天累计完成 180 分钟专注后才能兑换娱乐津贴。
- 周末没有工作时长门槛，但仍然需要消耗荣誉点。
- 积分没有负数下限；积分低于 0 或 60 分钟内累计扣 5 点会进入惩戒营。

**娱乐模式**

- 娱乐模式会话保留最近 30 轮临时上下文，停止后清除。
- 可设置 15 至 600 秒观察间隔。
- 可关闭 AI 点评；关闭后仍可计时和使用津贴。
- 如果识别到游戏和局势，AI 可以给出走位、技能、配装、资源或团队协作等战术建议。
- 娱乐点评只通过 AI 语音播报，不弹系统通知，也不写入近期观察。

## 可选 AI 配置

没有 API Key 时，应用仍可使用本地关键词、前台窗口规则和游戏识别。

如果要启用 AI，可以选择通用 OpenAI-compatible API 或本地 Ollama。

**OpenAI-Compatible API**

界面中可分别配置：

- 服务名称
- API Base URL
- API Key
- 文字模型
- 视觉模型
- 语音模型

文字模型负责文字判断、每日计划、文字证据审核和提醒措辞。视觉模型负责截图判断、截图证据审核和娱乐点评。语音模型留空时会停用 AI 语音相关功能。

API Key 只通过界面输入，并使用 Electron 系统安全存储加密；保存后输入框立即清空且不会回显，也不会写入普通设置文件。

**Ollama**

可以把文字判断、完成证据审查、提醒措辞和视觉任务优先分流给本地 Ollama。示例模型：

```powershell
ollama pull qwen3:8b
ollama pull qwen3-vl:8b
```

Ollama 模型不可用时，可选择回退到已配置的兼容 API。

## 可选 Cold Turkey 联动

- **\*注：本应用与Cold Turkey无利益关系，仅作为个人推荐功能加入，请自行斟酌使用，本人对相关风险概不负责。**
  
Cold Turkey Pro 不是必需组件。没有 Cold Turkey 时，专注计时、关键词监督、AI 判断、每日计划、荣誉值和娱乐津贴仍可使用。

Cold Turkey是一款功能强大的专注屏蔽软件，可以通过自定义屏蔽规则实现对网站、应用以及特定名称窗口乃至整台设备的屏蔽。
若希望使用此功能请自行准备Cold Turkey软件。

安装 Cold Turkey Pro 后，可以启用密码锁联动：

- 普通专注可锁定常规 Block，完成或证据通过后显示密码。
- 24 小时娱乐限制可锁定常规 / 娱乐限制 Block，兑换娱乐时显示临时解锁密码，娱乐结束后重新上锁。
- 惩戒营可使用单独的惩戒营 Block，默认名称为 `Games`，也可在界面修改。
- 常规 Block 和惩戒营 Block 建议不要使用同一个 Cold Turkey Block。

Cold Turkey 密码会使用 Windows 安全存储加密。最近 5 次成功使用的密码会额外保存在隐藏、限权且加密的本地恢复保险库中。

## 隐私与数据

- 本应用只在你主动开启专注或娱乐模式后读取前台窗口信息。
- 截图只在启用视觉判断、娱乐 AI 点评或粘贴计划证据时发送给你配置的模型服务。
- 截图不写入仓库，也不作为活动日志保存。
- 用户数据默认保存在 `%APPDATA%\ai-commissar`。
- API Key 不写入普通设置文件，不应出现在仓库或 release 包中。

分享给朋友时，不要打包你的 `%APPDATA%\ai-commissar` 目录，也不要共享包含 `.env`、`*.dat`、`settings.json`、`daily-plan.json` 或 `cold-turkey-session*.json` 的文件。
